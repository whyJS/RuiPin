  • Java程序设计
  • 学生通过本课程的学习掌握Java语 言编程规范和面向对象程序设计思想，旨在培养 学生分析问题、用Java语言编写程序解决实际问题的能力。

  • 课程简介

  • 关于本课

  • 本课程在讲解了Java语言基础知识和基本语法、多 线程、JDBC、典型通信程序、Servlet程序、 分布式编程等内容的基础上，进一步培养解题思维和编程 能力，即针对问题进行分析，理出算法并用Java语言 编程实现，从而为后续的学习和设计奠定必要的基础。

  • 前言

  • 配套教材：《Java语言及其应用》第一版，清华大学出版社，主编董佑平、夏冰冰

    参考资料： http://www.oracle.com

  • 先修知识要求

  • 1.《高级语言程序设计(C)》
    1. 掌握C语言的基本语法；
    2. 掌握结构化程序设计的基本思路和方法；
    3. 掌握高级语言程序编写、编译和执行的方法
    2.《面向对象程序设计(C++)》
    1. 掌握面向对象的基本理论；
    2. 掌握类和对象的定义方法，理解抽象数据类型的作用；
    3. 掌握Visual C++的基本操作和使用方法

  • 主讲教师

  • 夏冰冰

    董佑平
