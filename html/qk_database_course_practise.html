  • 青科2013级信息科学技术学院_数据库课程实训
  • 本课程实训通过项目实战， 掌握SQL Server 基本管理操作 、SQL Server数据库各对象的操作、 T-SQL脚本语言基本编程。

  • 课程简介

  • 关于本课

  • 本课程是针对《数据库》课程在学期末做的 课程实训（即数据库课程设计）， 主要是为了让同学们上完《数据库》课后 ，能够将理论应用到实际中，提高大家的动手能力。

    本课程通过一个实战案例《图书管理系统》， 让同学们体验开发中从设计到实现过程中， 与数据库有关的工作，主要包含了如下内容：

    1. 1、系统的需求分析
    2. 2、对分析结果进行设计
    3. 3、针对设计结果进行数据库的实现
    4. 4、各种业务功能的具体实现
    其中的第四步是我们练习的重点，希望大家重点关注。

  • 课程目录

  • 第01章
    • 第1节：图书管理系统数据库-需求分析
    • 第2节：数据设计的必要性
  • 第02章
    • 第一阶段，图书管理系统数据库-ER图-逻辑设计--物理设计
  • 第03章
    • 第1节：图书管理系统数据库创建及安全性设置
    • 第2节：图书管理系统数据库表的创建
  • 第04章
    • 第二阶段：图书管理系统表中测试数据的添加
  • 第05章
    • 第1节：完成员工管理模块业务
  • 第06章
    • 项目完结.演示得分
  • 第07章
    • 第三阶段：完成读者管理模块的业务
  • 第08章
    • 完成图书借阅模块的业务（一）
  • 第08章
    • 完成图书借阅模块的业务（一）
  • 第09章
    • 完成图书借阅模块的业务（二）
  • 第10章
    • 第四阶段：完成图书查询管理模块的业务
  • 材料汇总
    • 项目完结.演示得分
